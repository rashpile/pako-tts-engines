# pako-tts-engines

## Prerequisites

- Python 3.11+
- [uv](https://docs.astral.sh/uv/)

## Getting Started

```bash
# Install dependencies
make dev

# Run the application
make run
```

## Development

```bash
# Run tests
make test

# Run linter and formatter
make check
```

## Testing

```bash
make test
make test-coverage
```
